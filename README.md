# 经营驾驶舱 v6

给姜总/陆总看的经营利润驾驶舱。读 6 个数据源 → 按陆总《驾驶舱规则》算到税前利润 → 出一份自包含 HTML（科技风暗色默认、可切浅色、手机可看）。**前端不做任何金额运算**，所有数字 Python 算好。

> v1.3（亮色·两分区·业务线切换）已归档在 git tag `v1.3`；v6 是按陆总完整《驾驶舱规则》重构的版本。

## 跑

```bash
python run.py          # 生成 output/经营驾驶舱.html
sh tests/run_verify.sh # 语法 + 端到端 + 回归测试
```

## 数据怎么放

程序只读一个目录 `数据/`（`config.data_dir`）。**每月用新导出/新填的数据覆盖同名的 6 个文件即可**，文件名固定、见 `数据/README.md` 的来源对照。当前放的是 2026 年测试数据（当正式数据对待）；真实数据来了直接覆盖。
- `period_pin` = `null`：用系统当天年月（2026 数据 + 2026 今天）。要复现历史某月可临时填 `{year,month}`。
- 跑完看终端「数据体检」段 / 页面右上「数据体检」徽章：各源行数、月份覆盖、异常一目了然。

## 部署到新机器（git clone）

这不是常驻服务，是「读数据 → 生成静态 HTML」的批处理。步骤：

```bash
git clone https://github.com/EvanLee2004/BI.git
cd BI
cp 数据_样例/*.xlsx 数据/          # 首次可先用合成样例数据试跑
python run.py                      # 生成 output/经营驾驶舱.html
```

- **看**：直接双击 `output/经营驾驶舱.html`（自包含，离线也能看）。
- **要网址 / 手机访问 / 让「导出选文件夹」生效**：`python -m http.server 8000 --directory output`，浏览器开 `http://localhost:8000/经营驾驶舱.html`。（`file://` 双击方式下，导出会自动退成「下载一个 zip」，功能不受影响。）
- **正式用**：把真实数据覆盖进 `数据/` 的 6 个同名文件，再 `python run.py`。**`数据/*.xlsx`、`output/`、`data/*.json` 已被 `.gitignore`——真实数据/真实数字永不进库、不出本机。**
- **想每天自动更新**：给 `python run.py` 挂个每日定时任务（数据不会自己进来，得先有人/脚本把当天数据放进 `数据/`）。

## 一键导出（页面右上「⬇ 导出」）

点一下把当前看板导出到本地，便于后续财务分析 + 留档：
- Chrome/Edge（http 打开）：弹窗**选文件夹**，落 `经营驾驶舱数据_*.xlsx`（6 张 sheet）+ `经营驾驶舱快照_*.html` 两个文件。
- 不支持选文件夹（如 `file://` 双击）：自动**下载一个 zip**，内含上述两份，解压即用。
- 铁律：金额全 Python 侧算好嵌进页面，导出的 Excel 与页面逐格一致；前端只打包不算数。

## 6 个数据源（详见 `docs/数据来源说明.md` 与 `方案与文档/取数操作手册`）

| 源 | 供 | 类型 |
|---|---|---|
| 项目明细（智云·财务统筹） | 收入(交付额÷1.06) + 系统直接成本(项目成本) | 系统 |
| 任务·内部译员（智云·财务统筹·译员类型-资源库=IN-HOUSE） | 内部译员成本（从成本减） | 系统 |
| 下单（智云·财务订单管理） | 下单额 | 系统 |
| 回款记录（智云·FMS） | 回款额（只取到账、不看核销） | 系统 |
| 收单台账（财务·亮晶维护） | 市场/管理/固定运营/技术服务/财务费用 | 台账 |
| 手填与调整_月度.xlsx | 人力成本 + 生产成本手填项 + 财务费用补充 + 其他损益 | 手填(默认上月) |

## 口径（陆总 2026-07-03 定稿）

- 收入(不含税) = 交付额/本币 ÷ 1.06
- 生产成本 = 系统直接成本 − 系统内部译员成本 + 手填(PM/VM/实际内部译员/税费损失/技术流量/其他)
- 毛利 = 收入 − 生产成本
- 营销费用 = 营销人力成本(手填) + 市场费用(台账)；管理、研发同理；固定运营(台账)
- 财务费用 = 财务费用(台账) + 财务费用补充(手填)  ← 财务费用多为银行自动扣、台账显示不全
- 附加税费 = 收入 × 6% × 12%（系统自动算，和费用平行）
- 税前利润 = 毛利 − 营销 − 管理 − 固定运营 − 研发 − 财务 − 附加税费 + 其他损益(手填,默认0)

## 铁律（改代码勿破）

- **前端不做金额运算**：无 `toFixed/parseFloat/parseInt`（`tests` 有守卫）。
- 收单台账列按表头文字找、找不到即报错，不静默拿错列。
- 智云导出 xlsx 的 `<dimension>` 标签不可信，**绝不用 `read_only=True`**（详见 docs/数据来源说明）。
