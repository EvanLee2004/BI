# 数据目录 —— 每月把这 6 个文件放齐，程序就自动出驾驶舱

程序永远只读这个目录。**每月用新数据覆盖同名文件即可**（文件名固定不变）。

| 文件名 | 从哪来 · 怎么导 |
|---|---|
| `项目明细.xlsx` | 智云 → PMS/项目管理 → 项目明细（财务统筹视图，按整单交付日期，导出所有字段）→ 供 收入 + 系统直接成本 |
| `内部译员.xlsx` | 智云 → PMS/项目管理 → 任务（财务统筹视图，译员类型-资源库=IN-HOUSE）→ 供 内部译员成本（从成本里减） |
| `下单.xlsx` | 智云 → CMS/客户管理 → 订单 → 下单（财务订单管理视图）→ 供 下单额 |
| `回款记录.xlsx` | 智云 → FMS/财务管理 → 回款记录 → 供 回款额（只取到账，不看核销） |
| `收单台账.xlsx` | 财务收单台账（亮晶维护，服务器最新版）→ 供 市场/管理/固定运营/技术服务/财务费用 |
| `手填与调整.xlsx` | 自己维护的表：系统导不出的数（人力成本、生产成本手填项、财务费用补充、其他损益）。宽表，每月加一列，见表内「填写说明」页 |

> 详细取数步骤见 `../docs/取数操作手册`（或项目 方案与文档）。
> **当前放的是 2026 年测试数据（当正式数据对待）**；来了真实数据，直接覆盖同名文件即可。
> 每次跑完，看终端「数据体检」段 / 页面右上「数据体检」徽章，确认各源行数、月份覆盖、有无异常。
