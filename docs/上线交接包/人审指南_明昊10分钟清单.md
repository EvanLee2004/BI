## 0. 起服（约 1 分钟）

在终端：

```bash
cd 程序/看板正式程序
KANBAN_OFFLINE=1 KANBAN_FRONTEND=vue KANBAN_PORT=8018 .venv/bin/python run.py --serve
```

浏览器打开：

| 端 | 地址 |
|----|------|
| 看端 | http://127.0.0.1:8018/ |
| 管理端 | http://127.0.0.1:8018/admin |

**本机试用数据**是合成 golden，**不是真实客户金额**（看数字趋势/版式即可）。

默认试用账号（上线前必须改掉）：见 README「默认账号」——管理员 / 整体查看。

## 1. 看端（约 3 分钟）

| # | 页面 | 重点看什么 |
|---|------|------------|
| 1 | 登录 | 主按钮是**青色**（不是紫）；能登录 |
| 2 | 主页暗色 1440 | 五张 KPI 一排、间距整齐；字不糊 |
| 3 | 主页切浅色 | 右上角亮/暗切换；浅色也不刺眼 |
| 4 | 周期切换 | 能打开时间选择并切换（年/季/月）；页面不整页乱跳 |
| 5 | 下钻/排名 | 点一条排名或利润表大类，能弹出/展开，不白屏 |
| 6 | 手机 375 | 浏览器缩到手机宽，或真机连同一 Wi‑Fi 打开本机 IP:8018；按钮好点 |

**手机扫码怎么测**：电脑 `ifconfig` 看局域网 IP → 手机连公司/同一 Wi‑Fi → 浏览器打开 `http://<电脑IP>:8018/` → 登录同上。
