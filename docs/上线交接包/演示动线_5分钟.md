# 演示动线 · 5 分钟（给陆总首秀）

1. **总利润**（整体页 KPI）—「今年做到税前利润多少，一眼。」
2. **BU 利润表** — 点进某 BU：「条线自己的数，和整体一套口径。」
3. **期间费用构成** — 下钻大类/部门：「钱花在哪。」
4. **回款** — 「下单与回款对照，缺口在哪个月。」
5. **手机 375** — 扫码/缩窗：「出门也能看。」

## 她可能问的 5 个问题

1. **数字和 Excel 对得上吗？** — 同一套陆总口径；库算=文件算红线。
2. **工资为什么看不见？** — 产品决策全端并入「其他」，不单列。
3. **公共费用为啥 BU 是 0？** — 暂不分摊，不是 bug。
4. **谁能改数？** — 仅管理员；有调整台账可撤销。
5. **坏了找谁？** — 体检条 + 健康检查日志 + 部署执行单。
