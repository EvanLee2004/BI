# B-5 前端错误日志演示

1. 调用 `frontend_errors.record_frontend_error({"message":"B5_demo_intentional_error",...})`
2. 落盘：`数据/前端错误.log`（gitignore）
3. `frontend_error_stats` → yellow=True, count_24h≥1
4. `deploy/healthcheck.sh` 输出含 `YELLOW frontend_errors_24h=…`（见 healthcheck_with_fe_errors.txt）

端点：`POST /api/v1/client-error`（限流/截断/鉴权豁免评审：仅错误文本）
前端：`frontend/src/utils/frontendErrorReporter.ts` 挂 `boot-cockpit.ts`
