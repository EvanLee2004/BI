# 经营驾驶舱 · Windows 部署手册

> 目标机器：财务部自己的 Windows 电脑（陆经理要求：数据不出财务掌控范围）。
> 部署形态：**Python 定时生成静态 HTML + 可选内网端口**——没有数据库、没有常驻后端，机器重启后重跑一次 bat 即恢复。

---

## 一、装环境（只做一次，约 15 分钟）

1. **装 Python**：https://www.python.org/downloads/ 下载 3.10+ 安装包，
   安装时**务必勾选 "Add python.exe to PATH"**（第一屏下方的勾选框）。
2. **装依赖**（只有一个）：打开 cmd，执行
   ```
   pip install openpyxl
   ```
3. **拿代码**：二选一
   - 简单：把整个 `看板正式程序` 文件夹拷贝到部署机（U盘/内网传输均可）；
   - 或 git：`git clone https://github.com/EvanLee2004/BI.git`（仓库已私有，需要登录授权）。
   放哪都行，建议 `C:\看板\看板正式程序\`。**文件夹路径里不要带空格。**

## 二、放数据（每月一次的例行动作）

把 6 个源文件放进 `数据\` 目录，**文件名必须一字不差**（来源和导出步骤见 `数据/README.md`）：

| 文件 | 来源 |
|------|------|
| 项目明细.xlsx | 智云导出 |
| 下单.xlsx | 智云导出 |
| 回款记录.xlsx | 智云导出 |
| 内部译员.xlsx | 智云导出（任务表，按 IN-HOUSE 筛） |
| 收单台账.xlsx | 财务服务器台账原件（一年一 sheet） |
| 手填与调整.xlsx | 财务每月手填（宽表模板，每月加一列） |

## 三、生成报表

双击 **`更新看板.bat`**。程序会先跑"数据进门验证"：
- 全绿 → 生成 `output\经营驾驶舱.html`，双击就能看；
- 有 ✗ → **不会生成报表**，屏幕上写着哪个源、哪一列、Excel 第几行有问题，修好源文件重跑。

## 四、每天自动更新（任务计划程序）

1. 开始菜单搜"任务计划程序"→ 右侧"创建基本任务"。
2. 名称：`经营驾驶舱每日更新`；触发器：每天，建议 08:30；
   操作：启动程序 → 程序填 `C:\看板\看板正式程序\更新看板.bat`，
   "起始于"填 `C:\看板\看板正式程序`。
3. 建好后右键任务 →"运行"测试一次，确认 `output\` 里 HTML 的修改时间变了。
4. 注意：定时静默跑之前，把 `更新看板.bat` 末尾的 `pause` 那行删掉（否则窗口会停住等按键）。

> 说明：定时任务只是**重算**，数据本身还是二里那步人工放进去的。
> 所以现在的真实节奏是"每天自动重算 + 每月人工换数据"；要做到真·每日新数据，需要接智云 API（已列为下一步立项项）。

## 五、开内网访问（可选，⚠ 先过访问控制）

**开端口前必须先定访问控制方案（见下），未定之前只在本机双击 HTML 看。**

- 简易方式：双击 **`开启内网访问.bat`**（8018 端口，只共享 `output\` 目录，源数据不暴露）。
  同事访问 `http://机器IP:8018/经营驾驶舱.html`。窗口开着才有效，适合演示。
- 正式方式（推荐，带账号密码）：用 Caddy（单个 exe，免安装）：
  1. https://caddyserver.com/download 下载 caddy.exe 放 `C:\看板\`；
  2. 生成密码哈希：`caddy hash-password --plaintext 你要设的密码`；
  3. 在 `C:\看板\` 建 `Caddyfile`（无扩展名）：
     ```
     :8018 {
         root * C:\看板\看板正式程序\output
         file_server
         basicauth {
             finance 上一步生成的哈希串
         }
     }
     ```
  4. `caddy run` 启动（也可注册成 Windows 服务开机自启：`caddy run --config Caddyfile` 挂任务计划程序"登录时"触发）。
  同事访问时输入账号 `finance` + 密码。**分级展示 = 以后给不同人开不同账号/不同目录。**
- 防火墙：首次开端口 Windows 会弹防火墙询问，选"专用网络允许"；公司域环境可能需要 IT 开 8018 入站规则。

## 六、验收清单（部署完逐条打勾）

- [ ] 双击 `更新看板.bat` → 验证全绿 → HTML 生成、双击能看、有 logo；
- [ ] 故意把 `数据\收单台账.xlsx` 改个名再跑 → 报"文件不存在"且不出报表 → 改回来；
- [ ] 任务计划程序手动"运行"一次 → HTML 修改时间更新；
- [ ] （若开端口）另一台电脑/手机连公司 WiFi 能打开页面；关掉服务后打不开；
- [ ] （若用 Caddy）不输密码看不到页面。

## 七、已知边界

- 页面"⬇导出"按钮在 http:// 内网地址下没有"选文件夹"能力（浏览器安全限制，仅 https/localhost 支持），会自动走兜底：下载一个 zip（内含 Excel+页面快照），解压即用——这是设计内行为，不是故障。
- 台账里如有公式单元格且文件没在 Excel 里重新保存过，读出来可能是空值（openpyxl 读不到公式缓存）——台账请保持纯数值粘贴的习惯。
- 机器要常开（或至少工作时段开着），关机期间定时任务不跑、页面无法访问。
