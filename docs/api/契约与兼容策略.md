# API 契约与兼容策略（v1）

- **版本**：`/api/v1/*` 只加不删；破坏性变更升 `/api/v2`。
- **ViewModel**：`GET /api/v1/vm/cockpit`、`GET /api/v1/vm/bu/{name}` —— 语义键 + 后端显示串（`value_disp`/`pct_disp`/`*_html`）；前端零金额运算。
- **Legacy fragments**：`/api/v1/cockpit/fragments` 等保留一个版本期（阶段 3 默认 Vue 后标 deprecated）。
- **OpenAPI**：`/openapi.json`、`/docs` 仅管理员会话可见。
- **Schema 快照**：`tests/contracts/*.schema.json`；结构变更须显式更新快照文件。
