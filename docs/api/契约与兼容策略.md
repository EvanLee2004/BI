# API 契约与兼容策略（v1）

- **版本**：`/api/v1/*` 只加不删；破坏性变更升 `/api/v2`。
- **ViewModel**：`GET /api/v1/vm/cockpit`、`GET /api/v1/vm/bu/{name}` —— 语义键 + 后端显示串（`value_disp`/`pct_disp`/`*_html`）；前端零金额运算。
- **Legacy fragments**：`/api/v1/cockpit/fragments` 等保留一个版本期（阶段 3 默认 Vue 后标 deprecated）。
- **OpenAPI**：`/openapi.json`、`/docs` 仅管理员会话可见。
- **Schema 快照**：`tests/contracts/*.schema.json`；结构变更须显式更新快照文件。

## ViewModel 结构化字段（任务书50·B，只加不删）

| 端点 | 新增/强调字段 | 说明 |
|------|----------------|------|
| `GET /api/v1/vm/cockpit` | `kpi.cards_by_period` | 五卡显示串/目标条/BU 进度（废弃仅依赖 `body_by_period` HTML） |
| 同上 | `pl.table_by_period` | 利润表行 + 两级下钻 `details` |
| 同上 | `rankings.profit_rank_by_period` | 收入结构双卡结构化（`profit_rank_body` HTML deprecated） |
| 同上 | `expense.views_by_period` / `donut_center_by_period` / `*_y_axis_ticks` | 四态横条、环形中心、轴刻度显示串 |
| 同上 | `daily` | 日查询默认起止 |
| `GET /api/v1/vm/bu/{name}` | 同上结构 + `bu_names` | BU 页；桥接不向 BU 开整体接口 |
| **`GET /api/v1/vm/ledger`** | `columns/rows/total` | **任何会话（含管理员）白名单列**；禁止键见 `forbidden` |

- `body_by_period` / `daily_html` / `profit_rank_body`：**deprecated**，legacy 仍可读；Vue 真组件走结构化字段。
- 兼容：v1 契约只加不删；golden 数字/32 周期红线不因字段追加而变。
