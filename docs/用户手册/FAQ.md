# 常见问题 FAQ

**这篇解决什么问题**：数字不对、打不开、黄条吓人时，先自己查一遍再找人。

---

### 1. 打不开网页怎么办？

- **公司里**：只用内网 `http://192.168.30.46`（不要用外网地址试，办公区外网入口打不开是正常的）。  
- **公司外 / 手机流量**：用 `http://101.254.102.94:8001`。  
- **全员同一登录页（会话为单一 cookie，升级后无需记多套登录）**（2.5.0）：打开根地址登录即可；管理员与业务线共用一个登录，**没有**单独管理员登录网址。  
- 地址不要带一长串 `/bu/...`（登录后自动分流）。  
- **业务线账号第一次能进、关掉再开却进不去**：确认根地址（2.4.3 已加固）；仍不行找维护岗。  
- 管理端发版后白屏/点不动：强制刷新（Ctrl/Cmd+Shift+R）。  
- 仍不行：试电脑浏览器，找看板维护岗。

### 2. 登录提示账号或密码错误？

核对大小写；是否被连续输错锁定（等几分钟再试）。管理员可在设置里重置密码。

### 3. 顶栏黄条 / 体检黄是什么意思？

**2.2.8 起**：绿=本轮该抓的源都抓到且无业务提醒；**黄=抓齐了，但仍有业务提醒**（例如手填缺月、销售未归属 BU、调整过期、行数骤降警告）。点开体检详情看具体原因。不是系统坏了。

管理端若出现「⚠ ××本次未抓到」横幅，表示该源**这一轮没抓成功**、正在沿用本地文件——那种情况体检一般为**红**（见下条）。

### 4. 红条 / 体检红呢？

**2.2.8 起**：有「应抓」的数据源本次没抓到（智云某表/台账 local_fallback 或无源）、或登录冷却、或磁盘将满、或库检查失败等硬故障。先别依赖当天数字做重大决策，立刻找维护岗。关闭「智云自动抓」时，不会仅因智云未在线抓而判红。

### 5. 数字和 Excel 对不上？

常见原因：① 看板用的是**确认口径**（不是全部财务记账口径）；② 你看的时间段和 Excel 不一致；③ 手填/调整还没保存或未重算。先对齐「年/月」和业务线再比。

### 6. 为什么有的客户/销售在排名里是灰的「（未填）」？

智云下单没填部门等源头字段不全。管理员在 **异常处理 → 下单未填部门** 归类后，排名会正常。

### 7. 改完明细为什么刷新又变回去了？

若改的是源头 Excel 而不是管理端调整，下次抓数会覆盖。请用管理端「改数/调整」，才会留下调整记录。

### 8. 手机打开排版乱？

建议竖屏、系统浏览器；或换电脑。手机主要方便扫一眼，复杂操作建议电脑。

### 9. 只有自己业务线账号，为什么进不了别的业务线？

权限设计如此，避免串看。需要整体视角请用「整体」权限账号或找管理员。

### 10. 更新数据一直转圈？

等 2～3 分钟；看管理端更新状态。若一直失败，看黄条/日志，或检查智云/共享盘是否可达（部署机）。

### 11. 导出 Excel / HTML 失败？

看板顶栏「导出」下载的是 **静态可交互快照 HTML**（把当时的经营数据打进一个文件，用与线上同一套看板界面只读打开），**不是**需要一直连公司服务器的在线链接，也不是图片。双击下载的文件即可离线查看、切换年/季/月、展开利润表；整体账号导出的包可切换业务线，业务线账号导出的包只有自己。确认已登录且有权限（整体/管理员可导总页；BU 账号只能导自己的 BU）。明细导出 Excel 在白名单列范围内。换 Chrome/Edge 再试。

### 12. 谁可以改账号和密码？

仅**管理员**。普通人可在看端改自己的密码（若开放入口）；管理员在设置里管全部账号。

### 13. 公共费用分摊怎么填？

在 **数据调整 → 人工填写** 按月填各 BU 比例；合计不要超过 100%。保存后会重算。

### 14. 台账共享盘路径在哪改？

**设置 → 智云账号·台账路径**。改完保存；本机连不上共享属正常，部署机才必须通。

### 15. 还是解决不了找谁？

先把：① 账号类型（整体/BU/管理员）② 截图（可打码）③ 大概时间 发给**看板维护岗**（财务信息化）。不要把密码发到大群。

## 看板上看不到「工资」大类？

预期行为：工资全端并入「其他」，不单列（敏感）。详见使用手册「工资大类说明」。
