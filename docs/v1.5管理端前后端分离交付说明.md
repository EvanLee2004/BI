# v1.5.0-beta 管理端前后端分离 · 交付说明

基准：`393c04b` / v1.4.0-beta（含精简视图下线 + 顶栏 sticky）。  
原则：**搬家不装修**（无 React / npm；管理端观感与交互对齐内嵌版）。

## 拆了什么

| 原位置 | 现位置 |
|--------|--------|
| `server.py` `_ADMIN_CONSOLE` 内联 CSS | `static/admin/admin.css` |
| 内联应用 JS | `static/admin/admin.js`（磁盘含 `__MANUAL_ITEMS__` 占位） |
| 整页骨架 | `static/admin/admin.html`（`<link>` + `<script src="/admin/app.js">`） |
| `_BOOTSTRAP_HTML` | `static/admin/bootstrap.html`（首次取数引导） |

## 运行时怎么走

1. `GET /admin`（已登录且已取数成功）→ **只**返回 `static/admin/admin.html`  
2. 浏览器拉 `/static/admin/admin.css`、`/admin/app.js`  
3. `/admin/app.js`：读磁盘 `admin.js`，把 `__MANUAL_ITEMS__` 换成当前 `config.manual_items` JSON  
4. 未登录 → 登录页（小页仍可内嵌 `_LOGIN_HTML`）  
5. 未取数成功（`_state["admin_html"]` 空）→ `static/admin/bootstrap.html`  

## 已删除（2026-07-15 收口）

- `server.py` 内嵌整页 `_ADMIN_CONSOLE` / `_BOOTSTRAP_HTML`（约 100KB）**已删**  
- 管理端不再有「LEGACY 内嵌整页」双源；只维护 `static/admin/`  
- 回退整系统：仍可用 `git checkout <旧 commit>`（非管理端双源）  
- 看端：固定 shell+static（已去掉 `KANBAN_LEGACY_INLINE`；unittest 仍直出 HTML）
