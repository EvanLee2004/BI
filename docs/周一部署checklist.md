# 周一 Windows 部署 Checklist（v8.7.5 · 2026-07-13）

> 一页照做版，细节见 `Windows部署手册.md`。目标：财务部 Windows 机内网常开，`D:\看板\看板正式程序\`。
> **两件必验**（开发机验不了、部署机必须亲验）：① 看门狗起服务；② 一键更新真的 `git pull` + 重启。

---

## A. 装环境（约 20 分钟·联网·只做一次）

- [ ] 装 **Python 3.10+**，安装时勾 **Add python.exe to PATH**
- [ ] 装 **Git**（git-scm.com），`D:\看板\` 里 cmd 跑：
      `git clone https://gitee.com/Lee157/oracleeasy--bi.git 看板正式程序`
      （公开库、**不需要令牌**；clone 自 Gitee 则 origin=Gitee，一键更新默认对标 Gitee）
- [ ] 建环境装依赖（在 `D:\看板\看板正式程序\`）：
      `python -m venv .venv`
      `.venv\Scripts\pip install -r requirements.txt`
      `.venv\Scripts\pip install playwright`
      `.venv\Scripts\playwright install chromium`  ← 约150MB，智云自动登录必须

## B. 填配置（只做一次·都不进 git）

- [ ] ~~拷智云配置模板~~ **v8.7.5 起不用拷**：智云服务器地址+四表 ID 已内置在程序里（管理端「设置→智云账号→智云服务器与抓取表」可看/可改）；智云账号密码后面在管理端填
- [ ] **台账共享盘路径已内置默认**（亮晶共享盘·config.json 出厂值），**只剩智云账号密码在管理端填**（C 步起服务后）——`config.json` 全程保持不动（改它会让一键更新永久失效）；界面改的自动落 `数据\本地配置.json` / `数据\智云配置.json`（本机专属·不进 git）
- [ ] 首次起服务自动生成 `数据\管理员密钥.json`（无需手填）

## C. 起服务 → 界面填配置 → 首次取数（v8.7.5 新顺序）

> ⚠ 空机器直接 `python run.py` 会报"5 处缺文件不生成报表"=**正常**（数据还没抓），别在这一步纠结；照下面走。

- [ ] 双击 **`看门狗启动.bat`** 起服务（**必用它**，才支持一键更新自动重启；空机器能正常起、已在开发机模拟 clone 验证）
- [ ] 浏览器验：`http://<本机IP>:8018/`（用户端登录页）、`/admin`（管理端·lushasha/kanban2026）；同 WiFi 手机能开
- [ ] 防火墙首次询问选「专用网络允许」（域环境可能要 IT 开 8018 入站）
- [ ] 管理端「设置→🔑智云账号·台账路径」：填**智云账号密码**→保存（台账路径已预填默认、折叠区可核对四表 ID，一般不动）
- [ ] 管理端顶栏点 **「立即更新」** = 首次取数：预期智云三源 fetched（**内部译员在换专用账号前=兜底/体检黄 属正常**）+ 台账 fetched、页面出数，约 2~3 分钟；手填表没数只是警告（手填项按 0 计）

## D. 定时更新 + 开机自启

- [ ] 右键**以管理员身份**跑 `注册每日更新.bat`（按合并配置的 `schedule_times` 注册计划任务，默认 09:30；界面改过时间点也能读到——F-02 修复后走 loaders.load_config）
- [ ] **开机自启**：`看门狗启动.bat` 快捷方式放进 `shell:startup`（Win+R）

## E. 上线前必做（发账号前）

- [ ] 管理端「设置→👥账号与权限」：**把所有初始密码（8888 / kanban2026）改掉**
- [ ] 智云账号换成**正式/全量权限账号**（C 步先用什么号都行）→保存→点「立即更新」（内部译员应转绿）
- [ ] 管理端「设置→🏢BU 数据归属」：维护 BU 名/负责人/销售名单（拿到陆总映射定稿后勾选）
- [ ] 手填/部门预算目前是假数——**换真数后再把链接发陆总**（数据混合态别当真数给）

## F. 部署机必验两件（★开发机验不了）

- [ ] **看门狗**：确认服务由 `看门狗启动.bat` 起（不是直接 `run.py --serve`）；手动 `taskkill` 服务进程后看门狗能自动重拉
- [ ] **一键更新 pull+重启**：管理端「设置→版本与更新日志→检查更新」
      - 有远端时能查到状态（无新版本显「已是最新」+当前版本号）
      - 制造一个新版本（开发机 push 一小改到 Gitee）→ 部署机点「一键更新并重启」→ 应 `git pull --ff-only` 成功 + 服务以码 42 退出 + 看门狗用新代码拉起 + 页面自动刷新
      - 工作区脏 / 本地与远端分叉时应**拒绝**并提示人工（不会强拉覆盖）

- [ ] `数据\备份\` 每天多一份 `看板_YYYYMMDD.db`，超保留天数自动删最旧
- [ ] 自改密码闭环；管理端账号卡明文👁 / 加删改 / 初始密码黄标
