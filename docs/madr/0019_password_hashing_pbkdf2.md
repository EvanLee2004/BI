# MADR-0019：看板账号密码 PBKDF2-HMAC-SHA256 哈希存储

- **状态**：Accepted
- **日期**：2026-07-20
- **任务书**：63 · H-05

## 背景

任务书50 曾回退为明文（管理端 👁 可见），以便内网运维抄录。外部安全审查与明昊 2026-07-20 拍板：废除「管理员可查看密码」，改为「只能重置」；盘内只存哈希。

## 决策

采用标准库 `hashlib.pbkdf2_hmac('sha256', …)`：

- 格式：`pbkdf2_sha256$<iters>$<salt_b64>$<hash_b64>`
- 迭代次数：`600_000`（≥ NIST SP 800-132 常用量级）
- 每账号随机 salt 16 字节
- **零新第三方依赖**（离线内网部署不宜引 argon2/bcrypt 二进制轮子）

## 备选

| 方案 | 不取原因 |
|------|----------|
| argon2-cffi / bcrypt | 新依赖 + 编译轮子，离线/精简镜像成本高 |
| 仅 SHA256 无盐 | 不可接受 |
| 继续明文 | 与 H-05 产品拍板冲突 |

## 后果

- 读到旧「密码」明文 → 备份 `看板账号.json.bak-明文迁移-<日期>` 后写回哈希
- `/api/accounts` 永不下发密码；`POST /api/accounts/{账号}/reset_passwd` 明文仅响应一次
- 自改密码 `/api/my_passwd` 行为不变（验旧设新 + 密码版本+1）
