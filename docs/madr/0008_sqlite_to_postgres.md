# MADR-0008：SQLite → Postgres 触发条件与迁移路径

- **状态**：Accepted · 2026-07-17 · 任务书46·阶段5
- **背景**：当前 SQLite 单文件满足 ≤50 人内网读 + 日更一次写。
- **决策**：
  1. **不上 MySQL**（明确排除）。
  2. 触发迁 Postgres 条件（任一）：并发写冲突频繁、库文件 >2GB、需跨机高可用只读副本、合规要求独立 DB 服务。
  3. 迁移路径：`db.py` 抽象连接工厂 → SQL 方言层（占位符 `?`→`%s`）→ pgloader/自研导出导入 → 双写校验一周。
- **当前**：保持 SQLite；`db.py` 读写分离语义=读连接 URI `mode=ro`（见 connect 可选）。
