# 任务书 54.4 批次 D · 管理端 Vue（交付说明）

**日期**：2026-07-18  
**范围**：同一 `frontend/` monorepo 全量管理端 SPA；**未删** `static/admin`（legacy 回退 + 对照）。

## 怎么进管理端

| 模式 | 条件 | `/admin` 行为 |
|------|------|----------------|
| **vue**（默认有 dist） | `KANBAN_FRONTEND=vue` 或 config / 自动探测 dist | 吐 `frontend/dist/index.html`；前端 path 以 `/admin` 开头时 `bootAdmin()` |
| **legacy** | `KANBAN_FRONTEND=legacy` | 仍 static/admin + `/admin/app.js` |

深链：`/admin/settings`、`/admin/edit/manual` 等 → 同 SPA fallback（`admin_pages.admin_spa_fallback`）。  
**排除**：`POST /admin/login`、`GET /admin/logout`、`GET /admin/app.js`（精确路由优先）。

Cookie：`kanban_session` 不变；登录可用 `POST /admin/login`（form）或 `POST /api/v1/login`（JSON，Vue 登录页用后者）。

## 能力对表

| 能力 | Vue 视图 |
|------|----------|
| 控制台（iframe `/`） | `ConsoleView` |
| 更新数据 + 轮询 | `AdminLayout` |
| 手填 / 分摊 / 去税 | `ManualView` |
| 业绩目标 | `BudgetView` |
| 数据调整全表 | `DetailView` |
| 账号与权限 / BU / 设置 / 版本更新 | `SettingsView` |
| 异常总览 / 台账 / 下单部门 / 未分类 / 历史 / 审计 | `ExceptionOverview` 等 |

## static/admin 下线准备（未执行删除）

1. 默认 vue 路径已主用 Vue SPA。  
2. 删除前须：真浏览器走查写路径全绿 + `KANBAN_FRONTEND=legacy` 可删条款评审。  
3. 删除候选：`static/admin/admin.html|js|css`（**bootstrap.html 引导页可暂留**）。  
4. 回退开关：`export KANBAN_FRONTEND=legacy`。
