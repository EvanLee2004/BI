## ⑥ 提权固化（需你输一次 sudo 密码 · 建议做）

当前因 **无 NOPASSWD**，部署落在：

| 现状 | 目标（手册标准） |
|------|------------------|
| `/home/lee/kanban/看板正式程序` | `/opt/kanban/看板正式程序` |
| `systemctl --user` kanban | system `kanban.service` |
| `:8018` 直连 static | nginx `:80` + 反代 127.0.0.1:8018 |
| gvfs 用户挂载共享 | `/mnt/caiwu` CIFS + 本地配置 POSIX 路径 |

建议在机上执行（**你本地 `ssh -t kanban-home` 输 lee 密码**）：

```bash
# 1) 目录
sudo mkdir -p /opt/kanban && sudo rsync -a /home/lee/kanban/看板正式程序/ /opt/kanban/看板正式程序/
sudo chown -R lee:lee /opt/kanban

# 2) system unit（改 WorkingDirectory/User=lee）
sudo cp /opt/kanban/看板正式程序/deploy/linux/kanban.service /etc/systemd/system/kanban.service
# 编辑 User/Group/路径后：
sudo systemctl daemon-reload && sudo systemctl enable --now kanban
systemctl --user disable --now kanban   # 避免双开

# 3) nginx
sudo cp /opt/kanban/看板正式程序/deploy/linux/nginx-kanban.conf /etc/nginx/sites-available/kanban
# sed 路径后 enable + nginx -t && reload

# 4) 用户会话常驻（若仍用 user systemd）
sudo loginctl enable-linger lee

# 5) 可选 CIFS 正式挂载（凭据手填，不进 git）
# 见 docs/Ubuntu部署手册.md §4
```
